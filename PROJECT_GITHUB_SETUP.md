## How to Update:

1. **Replace with your actual GitHub URLs** in `frontend/src/components/Projects.js`:

```javascript
const projectsData = [
  {
    id: 1,
    title: "Word Document Clone",
    description: "...",
    tech: "React, Node.js, MongoDB, Express",
    github: "YOUR_ACTUAL_GITHUB_URL_HERE"  // ← Update this
  },
  // ... other projects
];
```

2. **GitHub URL Format:**
```
https://github.com/YOUR_USERNAME/YOUR_REPOSITORY_NAME
```
