# Image Setup Instructions

To add images to your portfolio, place the following files in the `public` folder:

## Required Images:

### Profile Image
- **File**: `public/profile.jpg`
- **Size**: 300x300px (recommended)
- **Description**: Your professional headshot for the About section

### Project Images
- **File**: `public/project1.jpg` - Word Document Clone
- **File**: `public/project2.jpg` - Smart Loan System
- **File**: `public/project3.jpg` - E-Commerce Price Prediction
- **Size**: 400x200px (recommended)
- **Description**: Screenshots or mockups of your projects

### Achievement Images
- **File**: `public/achievement1.jpg` - Best Developer Award
- **File**: `public/achievement2.jpg` - Hackathon Winner
- **File**: `public/achievement3.jpg` - Open Source Contributor
- **Size**: 300x250px (recommended)
- **Description**: Photos of certificates, awards, or achievement moments

### Resume
- **File**: `public/VarshaCV.pdf`
- **Description**: Your resume PDF file

## File Structure:
```
public/
├── profile.jpg
├── project1.jpg
├── project2.jpg
├── project3.jpg
├── achievement1.jpg
├── achievement2.jpg
├── achievement3.jpg
└── VarshaCV.pdf
```

## Image Guidelines:
- Use high-quality images (at least 300 DPI)
- Optimize file sizes (under 500KB each)
- Use JPG format for photos, PNG for graphics
- Ensure images are professional and relevant
